## Focus this week

Week 02 of logging my journey woohoo.. This one was heavy on devops. I've been wanting to get better at it out of pure curiosity and honestly just to level up.

## AWS

So for a change the protagonist of this story (me) decided to actually deploy something on AWS. The project is close sourced for now but once I feel it's ready I'll share it here ofc.

### EC2

First time spinning up an EC2 instance in a while and wow you forget how many little things there are — security groups, key pairs, making sure you're not accidentally leaving port 22 open to the entire internet. Got the instance running, SSH'd in, and started setting up the environment. It's one of those things that feels tedious until it clicks and then you're like okay this is kinda powerful.

### S3

Also set up S3 for storage. Nothing crazy, just getting buckets configured with the right access policies. The IAM permission stuff can be a rabbit hole but it's worth understanding properly especially when you're handling anything even remotely sensitive.

### Scaling plans

So I haven't deployed in a while so this was good practice. I want to scale up this system — hopefully I'll get some real users — but I'm mostly scaling it up for learning purposes. Load balancers, auto scaling groups, maybe even mess with CloudFront down the line. We'll see.

## The project

I intend to sell it as an internal tool to my university. Can't say too much about it yet but I'm making sure it's safe and following security standards because student safety is priority. No cutting corners on that.
